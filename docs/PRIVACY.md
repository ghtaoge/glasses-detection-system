# 隐私与使用边界

本系统在本机处理人脸图片，不调用云端视觉识别服务。数据集原图、标注、检测历史和模型均保存在配置的本地数据目录中。删除历史记录会同时删除对应原图、结果图和数据库记录。

摄像头视频帧只用于实时推理，不写入数据库或磁盘。用户点击“保存快照”后，服务端会重新验证并推理当前帧，再保存到历史记录。

系统只判断 `未戴眼镜`、`普通眼镜`、`墨镜`，不进行身份识别，也不应据此推断健康、能力或身份属性。禁止用于执法、就业、门禁、医疗、保险等高风险自动决策。

内置公开数据源采用 CC BY-NC 4.0，仅限非商业教学使用。导入前必须确认许可，分发数据或训练产物前应重新检查来源条款和肖像隐私要求。
